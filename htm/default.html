第一步：创建模板

       在项目的htm文件夹下创建一个html格式的html片段(不能包含html,head,body标签),

如：

       test2.html:

       <div> test2…….</div>

 

第二步：创建模板的编译和控制文件

       在项目js/controller目录下创建一个与html模板一样名字的js文件格式如下,类名不修改

       test2.js文件：

 

       App.controller = Ambow.extend(App.Controller,{

              tplName:'test2'

});

new App.controller();

 

以上是一个最简单的空白页面，其中tplName必须，是要载入的模板文件名

                                  

第三步：将模板加载到内容区(相当于多页面的跳转，以下用跳转来代替)

      

在js中添加跳转控制代码：App.History.load(‘test2’);

如我们点一个工具栏的button跳到这一样可以这样写：

//额外自定义工具栏按钮，没有可以不写

           toolBarExtra:[

                

                 {

                        id:'btntest2',

                            text:'测试2',

                            handler:function(){

                                   App.History.load('test2');

                            }

                 }

       ]

 

如此一个有模板，有控制，有跳转的简单页就完成了。
